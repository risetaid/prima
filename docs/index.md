## Project Overview

Patient Reminder and Information Management Application (PRIMA) focused on patient workflows, reminders, educational content, and admin operations.

## Quick Reference

- **Tech Stack:** Next.js 15, React 19, TypeScript, Drizzle ORM, PostgreSQL, Clerk, Tailwind
- **Entry Point:** `src/app/layout.tsx`
- **API Surface:** `src/app/api/**/route.ts`
- **Architecture Pattern:** Layered monolith (routes -> services -> data)
